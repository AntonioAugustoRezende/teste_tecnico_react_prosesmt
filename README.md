## Instalando e rodando o projeto

```bash
# Instale as dependências:
$ npm install

# Execute a aplicação:
$ npm run dev

# Acesse pelo seu navegador:
$ http://localhost:5173/
```

## Funcionalidades

### Gerais

- Visualizar os dados de todos os estados do país
- Filtrar por estado
- Filtrar por país
- Filtrar por data
- Adicionar novos dados
